# retail-store-discount-using-js
A very simple retail store discount system using Javascript and JQuery

## Installation

```js
npm install
```

## Run

```js
npm run build
```
Open the index.html to see the UI preview

## Test case and coverage

```js
npm run test -- --coverage
```
